Front End Development

In this section I would showcase one of my sample work for front end design, where I utilize JS, Anime, HTML coding practicese, CSS styling and bootstrap. An excercise tracker app, where I tested my ability to build an application on fullpage.js, while dabbling on animation on svgs, html elements and etc.

Excercise Tracker is a website which helps users to track on their progress of certain excercises built into the application. Once they've meet the neccessary requiment for the current exercise, a new one would be unlocked for them to keep track. Before executing and creating the website itself, I've first planned and design the user flow and layout of the website. Starting from the landing page where the user would use to input there username. Afterwards, I design how the homepage would look like and how to display the choices of activities along with the navigation and footer. Next the input page where users enter the ammount of time they took for doing they exercises for the week along with the page showing the graphs.
